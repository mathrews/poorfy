O que é o Poorfy:

  • É gratuito?

    Exatamente! Este site é gratuito para todos que consomem a lista imensurável de músicas presentes no site.

  • Quantas músicas?

    Neste site será apresentado qualquer tipo de música possível. Qualquer estilo presente no mercado estará nesse player.

  • É acessível?

    O aprimoramento e desenvolvimento contínuo do site é feito para a maioria das pessoas com qualquer necessidade ter acesso ao site.
